This project is a basic keylogger created for cybersecurity education. It logs every keystroke typed and saves them to a file called `keylog.txt`.

 Ethical Warning
This tool is for **educational use only**. Never run or deploy keyloggers on other people's devices without permission.

 Features
- Logs every key pressed
- Saves data into `keylog.txt`

 How to Use

 Run the script:

python keylogger.py
